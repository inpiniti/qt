# Installation

You can install React Query via NPM, Yarn, or Pnpm.

## NPM

```bash
npm i @tanstack/react-query
```

## Yarn

```bash
yarn add @tanstack/react-query
```

## PNPM

```bash
pnpm add @tanstack/react-query
```

## Requirements

- React v16.8+ (Hooks are required)
- Browsers: Modern browsers (Edge, Chrome, Firefox, Safari, Opera) and IE11 (with polyfills).

## Optional: ESLint Plugin

To enforce best practices, you can install the ESLint plugin:

```bash
npm i -D @tanstack/eslint-plugin-query
```
