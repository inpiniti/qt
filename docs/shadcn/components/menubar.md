# Menubar

Menubar 컴포넌트에 대한 문서입니다.

## Installation

CLI를 사용하여 프로젝트에 컴포넌트를 추가합니다.

```bash
npx shadcn@latest add menubar
```

## Usage

```tsx
import { Menubar } from "@/components/ui/menubar"

export default function MenubarDemo() {
  return (
    <Menubar />
  )
}
```
