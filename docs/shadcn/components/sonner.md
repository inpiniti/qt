# Sonner

Sonner 컴포넌트에 대한 문서입니다.

## Installation

CLI를 사용하여 프로젝트에 컴포넌트를 추가합니다.

```bash
npx shadcn@latest add sonner
```

## Usage

```tsx
import { Sonner } from "@/components/ui/sonner"

export default function SonnerDemo() {
  return (
    <Sonner />
  )
}
```
