# Slider

Slider 컴포넌트에 대한 문서입니다.

## Installation

CLI를 사용하여 프로젝트에 컴포넌트를 추가합니다.

```bash
npx shadcn@latest add slider
```

## Usage

```tsx
import { Slider } from "@/components/ui/slider"

export default function SliderDemo() {
  return (
    <Slider />
  )
}
```
