# Input Group

Input Group 컴포넌트에 대한 문서입니다.

## Installation

CLI를 사용하여 프로젝트에 컴포넌트를 추가합니다.

```bash
npx shadcn@latest add input-group
```

## Usage

```tsx
import { InputGroup } from "@/components/ui/input-group"

export default function InputGroupDemo() {
  return (
    <InputGroup />
  )
}
```
