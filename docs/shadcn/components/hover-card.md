# Hover Card

Hover Card 컴포넌트에 대한 문서입니다.

## Installation

CLI를 사용하여 프로젝트에 컴포넌트를 추가합니다.

```bash
npx shadcn@latest add hover-card
```

## Usage

```tsx
import { HoverCard } from "@/components/ui/hover-card"

export default function HoverCardDemo() {
  return (
    <HoverCard />
  )
}
```
