# Native Select

Native Select 컴포넌트에 대한 문서입니다.

## Installation

CLI를 사용하여 프로젝트에 컴포넌트를 추가합니다.

```bash
npx shadcn@latest add native-select
```

## Usage

```tsx
import { NativeSelect } from "@/components/ui/native-select"

export default function NativeSelectDemo() {
  return (
    <NativeSelect />
  )
}
```
