
# Metadata Files

Special files to define metadata.

## Icons
- `favicon.ico`, `apple-icon.jpg`, `icon.png`: Set favicon and icons.

## Open Graph / Twitter
- `opengraph-image.png`, `twitter-image.png`: Set OG images.

## SEO
- `sitemap.xml` / `sitemap.ts`: Generate sitemap.
- `robots.txt` / `robots.ts`: Generate robots.txt.
