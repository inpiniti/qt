
# Next.js MCP Server

A specialized Model Context Protocol (MCP) server integration for Next.js projects.

## Overview
This server provides AI assistants with deep context about the Next.js project structure, routing, and conventions.

- **Resource Listing**: Automatically lists pages, components, and API routes.
- **Context Awareness**: Helps AI understand App Router vs Pages Router conventions.
- **Tooling**: Provides specific tools for Next.js specific tasks (e.g., generating metadata, verifying route config).

*(Note: specific implementation details would depend on the actual MCP server package used)*
