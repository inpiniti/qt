
# Migrating to v5 from v4

Zustand v4에서 v5로 마이그레이션 (예상 변경 사항).

## 현재 상태
Zustand v5는 아직 베타 또는 개발 단계일 수 있습니다. (문서 작성 시점 기준)

## 잠재적 변경 사항
- **React 18/19 지원 강화**: `useSyncExternalStore` 활용 최적화.
- **API 정리**: 더 엄격한 타입 추론 및 더 나은 미들웨어 지원.
- **Deprecated 제거**: `zustand/context` 등 오래된 API 완전 제거 가능성.

공식 릴리즈 노트를 확인하세요.
