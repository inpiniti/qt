# 큐티 작성 워크플로우

사용자가 본문 범위, 유튜브 링크, 대상 정보를 제공하며 큐티 작성을 요청할 때 이 워크플로우를 실행합니다.

## 단계

1. **정보 수집 및 분석**:
   - 사용자가 제공한 본문 범위, 유튜브 링크, 대상 정보를 확인합니다.
   - 유튜브 링크의 내용을 파악하기 위해 (필요시) 관련 정보를 검색하거나 분석합니다.

2. **큐티 내용 생성**:
   - `qt-mentor` 스킬의 가이드라인에 따라 내용을 작성합니다.
   - 문학 소년의 감수성과 신앙 멘토의 통찰력을 담아 서정적으로 집필합니다.

3. **파일 저장**:
   - 생성된 내용을 `qt/` 폴더에 마크다운 파일로 저장합니다.
   - 파일명 형식: `YYYY-MM-DD_[주요키워드].md`
   - 위 워크플로우를 실행할 때 `qt/` 폴더가 없다면 먼저 생성합니다.

4. **결과 보고**:
   - 생성된 파일의 경로와 주요 내용을 사용자에게 안내합니다.
